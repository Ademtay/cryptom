# 🚀 CryptoTracker Pro

## 📊 Proje Açıklaması
CryptoTracker Pro, kripto para piyasalarını gerçek zamanlı takip etmenizi sağlayan modern bir web uygulamasıdır. Kullanıcı dostu arayüzü ve gelişmiş analiz araçlarıyla, kripto yatırımcıları için vazgeçilmez bir asistan olarak tasarlanmıştır.

## ⭐ Temel Özellikler
- 💹 Gerçek zamanlı fiyat takibi
- 📈 Detaylı grafik analizleri
- 🔔 Fiyat alarm sistemi
- 📱 Mobil uyumlu tasarım
- 🔄 Otomatik portföy güncelleme
- 🔐 Güvenli API entegrasyonu

## 🛠️ Kullanılan Teknolojiler
- **Frontend:** React.js, TailwindCSS
- **Backend:** Node.js, Express
- **Veritabanı:** MongoDB
- **API:** CoinGecko, Binance API
- **Güvenlik:** JWT, bcrypt
- **Deployment:** Docker, AWS

## 📦 Kurulum
```bash
# Repository'yi klonlayın
git clone https://github.com/kullaniciadi/cryptotracker-pro.git

# Proje dizinine gidin
cd cryptotracker-pro

# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```

## 🔧 Konfigürasyon
`.env` dosyasını oluşturun ve aşağıdaki değişkenleri ekleyin:
```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
COINGECKO_API_KEY=your_api_key
BINANCE_API_KEY=your_binance_api_key
```
